The Te Urewera Hunting webmap is designed to be a simple, responsive web mapping application with added features.

Features
  • Fullscreen pc, tablet & mobile-friendly map with built-in compatibility for multiple browsers.
  • A variety of data sources are used from live government links to customised local data converted to GeoJSON and in some cases made available for downloading in Google Maps/Earth compatible format.
  • Points of Interest are easily searchable and their markers are grouped at low zoom level with a number to indicate quantity in that vicinity. Markers disperse at higher zoom levels.
  • Elegant multi-layer feature search with autocomplete.
  • Responsive sidebar feature list synced with current map display, which includes sorting and filtering.
  • Location of user available when GPS is on and available.
  • Important note for Printing:
    No built-in print function. If you require a printed map try using Google Chrome browser, it has the most compatible printing function for javascript based web pages.

The data provided on this site is for informational and planning purposes only.
